# conte-informatica
Static commercial landing page optimized for responsiveness, organization, and performance using pure HTML, CSS, and JavaScript.

## 🖥️ Computer Store Landing Page

This project is a landing page developed for a computer store to present products and services in a clean, modern, and responsive way.

### 🎯 Purpose
Build a lightweight and visually appealing institutional page while applying solid front-end best practices.

### 🛠️ Technologies
- Semantic HTML5
- Modern CSS3 (Flexbox and responsive layout)
- Vanilla JavaScript for basic interactions

### 💡 Technical Highlights
- Semantic structure focused on SEO
- Fully responsive layout for mobile, tablet, and desktop
- Clean and maintainable code organization
- Performance-oriented approach without frameworks

### 📱 Responsiveness
Designed to work seamlessly across different screen sizes and devices.
